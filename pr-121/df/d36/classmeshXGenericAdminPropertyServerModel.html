Detailed Description

A template class for creating Generic Admin Property Server models.

This class is derived from meshXServerModel and provides a convenient interface for creating Generic Admin Property Server models. It handles the Generic Admin Property state change notifications from the MeshX stack and publishes the state change event to the element layer.
