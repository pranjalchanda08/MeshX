1/**
2 * @file meshx_model_property.hpp
3 * @brief Implementation of Generic Property Models for MeshX
4 *
5 * This file contains the implementation of the Generic Property models,
6 * which provide standard Property model functionality in the MeshX BLE mesh framework.
7 *
8 * Key Features:
9 * - Implements Bluetooth SIG-defined Generic Property models
10 * - Supports Manufacturer, Admin, User, and Client Property servers
11 * - Inherits from meshXClientModel and meshXServerModel templates
12 * - Provides standard Property control operations (GET, SET operations by property ID)
13 * - Integrates with MeshX transmission control
14 *
15 * @author Pranjal Chanda
16 * @date 2024-2025
17 * @copyright Copyright 2024 - 2025 MeshX
18 */
