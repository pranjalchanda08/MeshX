Send a CW/WW (Cool White/Warm White) message over BLE Mesh.

Parameters
[in]pdevPointer to the device structure.
[in]model_idModel ID of the CW/WW client.
[in]element_idElement ID to which the message is addressed.
[in]set_getFlag indicating whether the message is a set (1) or get (0) operation.
[in]is_rangeFlag indicating whether the message is a temperature range set (1) or not (0).
[in]ackFlag indicating whether the message requires an acknowledgment (1) or not (0).
Returns
  • MESHX_SUCCESS: Success
  • MESHX_INVALID_ARG: Invalid argument
  • MESHX_FAIL: Sending message failed
