Detailed Description

Common definitions for BLE Mesh models and opcodes in the MeshX framework.

Copyright (c) 2024 - 2025 MeshX

This header file contains macros and definitions for BLE Mesh models, opcodes, and related constants used in the MeshX framework. It includes:

  • Address type checks and constants.
