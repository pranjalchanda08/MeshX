Detailed Description

Implementation of the MeshX generic server model for BLE mesh nodes. This file contains functions for registering, deregistering, and initializing the generic server model.

Copyright (c) 2024 - 2025 MeshX

The MeshX generic server model provides an interface for handling BLE mesh server operations, including callback registration and initialization.

Pranjal Chanda

Sends a status message for the Generic Server model.

Returns
